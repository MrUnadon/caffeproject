01.畑にきた新しいミョウレンジャー

　

こんにちは、加藤です！　　ほりこしだよー！

　

(加藤) 本日はお日柄もよく、GWで唯一の奇跡の快晴ということで、コンニャクヤ畑に表敬訪問にきました。市民の生活に触れるのは、市長の重要な務めです。

　

(ほりこし) 今日は関東の畑の勉強をしにきたよー。仙台にも畑あるからおいでー。

　

(加藤) 市民の生活模様はいかがでしょうか。商品価格には、その土地の人々の暮らしが反映されます。さっそく、生産者に野菜の値段を聞いてみましょう。

　

　

　

　

(加藤) だめだ。早くなんとかしないと。

　

02.葉野菜の収穫

　　

(ほりこし) へぇ〜〜。すごいね！野菜が育ってる！え〜っ！これ「ほりこす」いるじゃん！収穫していいの？

　

(加藤) 結構量あるね、まあでもたくさん人いるし

　

(いけだ) もうどんどんとっちゃって！ 今日はいっぱい人がくるから、全部収穫しよう！

　

(ほりこし) どうすんの？軍手貸して？笑

　

(ひらが)　やさしく収穫するんだよ、ほら💙

　

　

(いけだ) これ以上うえててもあれだから、水菜もほうれん草も、たくさん持ってきな！

　

(さかもと) いいタイミングで育ったね！最初はあんなに小さなタネだったのに！

　

　

03.買い出し班

　

(あすか) ねえ、今日なにすんの？

　

(ひらが) きょうは野菜を食べてお祝いをするんだよ、あすかちゃん

　

(あすか) 野菜は何になるの？サラダ？あとパイナップルの食べ比べしたい！

　

(いけだ) それはシェフに聞いてw パイナップルもシェフに言って！

　

(ひらが) あいつに聞かないと…俺たちだけだと、野菜は全部…葉っぱだからさ

　

　

(いけだ) まぁ何になるかはよしとして、酒もバッチリ買い込んだし、あとは主賓のお二人を待ちますか！

　

04.なかよし野菜パーティー

　

おおおかくん。さなえちゃん。いらっしゃい🎉

今日は朝早くから疲れてるところ、ありがとう！

　

　

　

(ひらが) 早速、物理的に駆けつけてくれたね 笑

　

(さなえ) 特技だからね！どこにでもお呼びなさい？

　

(ひらが) あれ、おおおか氏は、トイレかな？

　

　

(おおおか) さっきから ここにいるけどw

　

　

(おおおか) さかもとくん、それイサキ？

　

(さかもと) そうそう！白ワインのソースを合わせようと思って。ちょっとまっててね！ブレンダーとか無いから、みじん切りとかで、時間かかるけども。

　

(さなえ) ヴィーーーンゥゥ ヴィーーーン [ミキサーの真似]

　

　

　

(さかもと) おまたせ！

　

(あすか) ねぇパイナップルでなんかアイスみたいなデザートないの？ブルーベリーとか🍨
